```
Generate Go code based on the introspection of a GraphQL server

Usage:
  tutone [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  fetch       Fetch GraphQL Schema
  generate    Generate code from GraphQL Schema
  help        Help about any command

Flags:
  -c, --config string     Path to a configuration file (default ".tutone.yml")
  -h, --help              help for tutone
  -l, --loglevel string   Log level (default "info")
  -v, --version           version for tutone

Use "tutone [command] --help" for more information about a command.
```
